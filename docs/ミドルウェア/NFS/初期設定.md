## サーバ側
### インストール

```bash
yum -y install nfs-utils
```

### マウント元の許可設定

- /etc/exports
 
 ```conf
 /var/www/html 192.168.1.0/24(rw,async,no_root_squash)
 ```

### 設定反映

```bash
exportfs -ra
```

### チューニング
#### nfsのプロセス数変更(必要であれば)

- /etc/nfs.conf

```conf
[nfsd]
threads=32
```

### 起動

```bash
systemctl enable --now nfs
```

## クライアント側
### チューニング
#### マウントのバージョンの切り替え

- /etc/nfsmount.conf

```conf
#Defaultvers=4
Defaultvers=3
```

### mount

```bash
/bin/mount -t nfs 192.168.1.1:/var/www/html /var/www/html
```

### マウント状況の確認

```bash
nfsstat -m
```
