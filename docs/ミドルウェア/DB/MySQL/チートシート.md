## 作成

### ユーザ作成

```sql
create user [ユーザ名] identified by '[パスワード]';
```

### データベース作成

```sql
create database [データベース名];
```

---

## 更新

### パスワード更新

```sql
alter user [ユーザ名] identified by '[パスワード]';
```

### ユーザーの権限更新

```sql
grant all on [データベース名].* to [ユーザ名];
```

---

## 削除

### キャッシュ削除（権限反映）

```sql
flush privileges;
```

---

## 参照

### ユーザ・ホスト・認証情報の確認

```sql
select user, host, authentication_string, plugin from mysql.user;
```

### 権限の確認

```sql
show grants for [ユーザ名];
```

### カラム情報の確認

```sql
show columns from [テーブル名];
```

### SQL モードの確認

```sql
select @@GLOBAL.sql_mode;
```

### 現在接続しているデータベースの確認

```sql
select database();
```

### 現在の接続数

```sql
show status like 'Threads_connected';
```

### 最大接続数（設定値）

```sql
show global variables like 'max_connections';
```

### 起動後の最大接続数（統計）

```sql
show global status like 'Max_used_connections';
```

### 実行中のクエリ一覧

```sql
show full processlist;
-- または
select * from information_schema.PROCESSLIST;
```

### 10秒以上経過しているクエリの確認

```sql
select * from information_schema.PROCESSLIST where time >= 10;
```

## バックアップ

### 全データベースのバックアップ

```bash
mysqldump -u [ユーザ名] -p -h [接続先ホスト] --single-transaction -A > dumpall.sql
```

### 特定のデータベースのバックアップ

```bash
mysqldump -u [ユーザ名] -p -h [接続先ホスト] --single-transaction [データベース名] > database.sql
```
